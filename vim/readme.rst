PHP Debugging
-------------

* install ``php-xdebug``
* ``phpenmod xdebug``
* add these lines to your ``php.ini``

  ::
      [xdebug]
      xdebug.remote_enable=1
      xdebug.remote_host=localhost
      xdebug.remote_port=9000
